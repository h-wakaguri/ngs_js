fasta/gzi(Required same prefixed three files: bgzipped fasta .gz, .gz.gzi and .gz.fai)

After compressing the FASTA file with bgzip, processing the .gz file with samtools faidx will generate a .gzi file. If the FAI file is named input.fasta.fai, please rename it to input.fasta.gz.fai.

Please select .gz, .gz.gzi and .gz.fai files.
